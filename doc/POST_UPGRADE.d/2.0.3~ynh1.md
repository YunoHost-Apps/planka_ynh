Since 2.0.3~ynh1, the Yunohost package supports the default language setting used by Planka.

> The default application language used as a fallback when a user’s language is not set. This language is also used for card notifications.

If Planka was originally installed in an older version, this setting was set by default to `en_US` during the update. If you wish to modify this setting, please follow the detailed procedure below.

---

**How to change the default language after installation (for example to `fr_FR`):**

1. Change the app setting in YunoHost: `yunohost app setting planka language -v fr_FR`
2. Change the app setting in Planka: `sed -i 's/^DEFAULT_LANGUAGE=.*/DEFAULT_LANGUAGE=fr_FR/' /var/www/planka/.env`
3. Restart the planka service: `yunohost service restart planka`
