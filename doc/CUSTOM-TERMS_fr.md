### Personnalisation des conditions d'utilisation pour les utilisateurs finaux

Planka inclut un modèle par défaut de conditions d'utilisation pour les utilisateurs finaux.

Vous pouvez remplacer intégralement les conditions fournies par votre propre version personnalisée, mais la suppression complète des conditions n'est pas prise en charge. Vous pouvez toutefois remplacer le contenu par un message minimal de bienvenue destiné aux utilisateurs et, si vous le souhaitez, omettre la section des confirmations. Dans ce cas, le bouton « Continuer » sera actif par défaut et aucune case à cocher d'acceptation ne sera requise.

Documentation officielle (en anglais) : https://docs.planka.cloud/docs/configuration/customizing-end-user-terms/
